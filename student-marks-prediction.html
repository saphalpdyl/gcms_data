Processing repetitive values in some columns

Columns like stu_group and location have values that are deploy the same meaning but are different only in case. For example, in column stu_group, there are values such as science and Science.

Problem

When one hot encoding using, pd.get_dummies(), there will be more than necessary number of columns.

Solution

We are going to convert all values to lower case.

Generating the target

The best way to evaluate student performance is through their total percentage ( assuming each subject has a total marks of 100 ).

from sklearn.metrics import mean_squared_error, r2_score, explained_variance_score, max_error

# Evaluating
error_dfs = []
def train_and_generate_error(model):
    model.fit(X_train, y_train)
    y_pred = model.predict(X_test)
    
    mse = mean_squared_error(y_test, y_pred)
    r2 = r2_score(y_test, y_pred)
    explained_variance = explained_variance_score(y_test, y_pred)
    max_err = max_error(y_test, y_pred)
    
    error_df = pd.DataFrame([], columns=["Error Name","Error Value","Error Range"])
    
    def generate_error_row(error_name: str, err_val: float, range: str):
        error_df.loc[len(error_df)] = [error_name, err_val, range]
    
    generate_error_row("Mean Squared Error", mse, "[0, inf)")
    generate_error_row("R2 Score", r2, "[0, 1]")
    generate_error_row("Explained Variance", explained_variance, "")
    generate_error_row("Max Error", max_err, "(-inf, inf)")
    
    error_dfs.append(error_df)
